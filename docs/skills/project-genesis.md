# Skill: project-genesis

## Goal

初期世界観が未整備でも、Codexとの往復を開始できる最小構成を作る。

## Inputs

- `workspace_bundle.json`（starter でも可）
- `codex_task.md`（taskType: `project genesis`）
- `validator_report.md`

## Preferred response modes

1. `codex_questions.md`（要件確認）
2. `codex_ops.json`（最小差分）
3. `updated_workspace_bundle.json`（大規模更新）

## Checklist

- routes / stateAxes / revealPlans / branchPolicies の整合
- chapter.scenePackages の canonical 性維持
- unresolved の明記
- draft rebuild 要否の明記
